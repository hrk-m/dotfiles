## 初期設定

```bash
$ curl https://raw.githubusercontent.com/hrk-m/dotfiles/master/install.sh | sh
```

## その他設定

```bash
$ git clone git@github.com:hrk-m/dotfiles.git
$ cd dotfiles
$ sh set_symlink.sh
```
